# 개발다봐 (DevLooks)

### 1. 프로젝트 주제

모바일 기반 개발 관련 웹사이트 추천 및 사용자화 서비스

### 2. 개발 스택
- 클라이언트: Java, RxAndroid, Retrofit2, Jsoup, MVP 디자인 패턴 반영
- 서버: AWS EC2, PHP7 (Laravel Framework), MySQL

### 3. 기여 부분
- 외부 서버 콘텐츠 파싱 구조 설계
- 안드로이드 구조 설계
- 회원가입 기능 구축 (서버 통신)
- 검색 화면 기능 구축 (서버 통신)
- 콘텐츠 내부 화면 구축 (북마크 기능, 공유 및 내용 시각화)
